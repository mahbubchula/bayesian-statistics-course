# Assignments

The assignments build toward the final capstone. Each assignment should be submitted as a short report with code, figures, and interpretation.

## Assignment Sequence

| Assignment | Topic | Course Stage |
|---|---|---|
| 1 | Bayesian foundations | Weeks 1 to 3 |
| 2 | Conjugate models | Weeks 4 to 5 |
| 3 | Monte Carlo and MCMC | Weeks 5 to 6 |
| 4 | Hierarchical regression | Weeks 7 to 9 |
| 5 | Capstone project | Weeks 10 to 12 |

## Standard Submission Format

Submit:

- A report in PDF, Markdown, or notebook format
- Code used for calculations
- Figures or tables generated from code
- A short interpretation section
- A limitation or sensitivity-analysis section

## Standard Rubric

| Criterion | Points |
|---|---:|
| Correct model formulation | 25 |
| Correct computation | 25 |
| Interpretation and communication | 25 |
| Reproducibility and clarity | 15 |
| Reflection on limitations | 10 |
